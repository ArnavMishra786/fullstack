# 🤖 AI Legal Awareness Platform

An AI-powered legal awareness platform designed to help users understand their legal rights, laws, and procedures in a simple, interactive, and accessible way.

---

## 🚀 Features

- 🧠 AI-powered legal guidance
- 📚 Simplified explanations of laws
- 💬 Interactive chatbot for legal awareness
- 🌙 Modern dark UI with animations
- 📱 Responsive design

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **UI:** Tailwind CSS, Animations
- **AI:** NLP-based legal assistance (planned)
- **Version Control:** Git & GitHub
