# React Client Router Demo

This project demonstrates **client-side routing in React** using **React Router DOM**. It allows smooth navigation between multiple pages without reloading the browser.

---

## 🚀 Features

- Single Page Application (SPA)
- React Router DOM integration
- Multiple pages (Home, About, Contact)
- Clean navigation menu
- Fast page switching without reload
- Beginner-friendly project structure

---

## 🛠️ Technologies Used

- React.js
- React Router DOM
- JavaScript (ES6)
- HTML5
- CSS3
- Vite
