# User Registration Form (React)

## 📌 Project Description

This is a simple User Registration Form built using React (Vite).  
The form collects user details and displays them in an alert box upon submission.

This project demonstrates:
- React functional components
- useState hook
- Form handling
- Validation
- Basic UI styling with CSS

---

## 🚀 Features

- First Name input
- Last Name input
- Date of Birth (Future dates disabled)
- Gender selection (Radio buttons)
- Address field (Textarea)
- State selection (Dropdown)
- Skills selection (Checkboxes)
- Submit button (Displays entered data in alert)
- Cancel button (Resets entire form)

## ▶️ How to Run the Project

1. Navigate to the project folder:

cd exp6/myapp1


2. Install dependencies:

npm install


3. Start the development server:

npm run dev


4. Open the displayed local URL in your browser.

---

## ✅ Form Validation

- Date of Birth does not allow future dates.
- All fields are required.
- Cancel button clears all input fields.
